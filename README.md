# Mises Browser

Source-code for Mises Browser, a Web3 Browser built on top of [Chromium](https://www.chromium.org/Home/).


## Releases
Mises is currently built for Android and IOS

Official releases of Mises Browser Android can be downloaded from PlayStore

## Features
- Extension supports on android! And we have tested mises with about 150 web3 extensions
- A web3 navigator page that may help new web3 users
- Supports  the mises chain, a public chain focus on decentralized social network
- We made metamask  an default walllet on both android and ios version
